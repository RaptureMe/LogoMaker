# Logo Maker

## Description

Created a Logo Generator that takes any 3 letters and creates a svg file with a choice of shapes and colors.

## Table of Contents

- [Usage](#usage)
- [Credits](#credits)

## Installation

Install NPM to get Jest for testing, Node to command line Index.Js to create SVG file
